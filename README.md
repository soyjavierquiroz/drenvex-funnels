== Description ==
Cliente de captación para ecosistema Drenvex.
- Detecta /r/{referral_code}/{funnel_slug}
- (Próximos pasos) valida referral con CORE y envía leads

== Installation ==
1. Copia la carpeta drenvex-funnels en wp-content/plugins/
2. Activa el plugin desde el admin.
3. Configura CORE Base URL y CORE API Key en Ajustes -> Drenvex Funnels
